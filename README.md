# Yachay-Redescolar
YACHAY Escolar es una plataforma de mediación comunitaria. Se diseñó este espacio no solo como una herramienta técnica, sino como un servicio de gestión de información social. La plataforma permite conectar de forma ágil y segura a familias que disponen de materiales con aquellas que los necesitan, fomentando la economía circular.
